# Secondhand Car Scraper

Source: olx.co.id

## Dependencies
1. Python 3.6 or above
2. Selenium + Chromedriver
3. Beautifulsoap

## Usage
1. Export environment variable
  ```
  export CHROMEDRIVER_PATH="/path/to/chromedriver"
  ```
2. Run script
  ```
  python scraper.py
  ```
3. Output will be saved as csv file in `/output` dir
